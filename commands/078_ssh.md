## Detailed Flag Reference (from project flag table)

- `-p`: Specify the port to connect to on the remote host
- `-i`: Select the file from which the identity (private key) is read for authentication
- `-o`: Pass a configuration option in the format used by ssh_config
- `-N`: Do not execute a remote command; useful for port forwarding only
- `-L`: Forward connections from a local port to a specified remote host and port
