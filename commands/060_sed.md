## Detailed Flag Reference (from project flag table)

- `-i`: Edit files in place; GNU requires no argument, BSD requires a backup suffix (use '' for none)
- `-E`: Use extended regular expressions (POSIX ERE); portable across GNU and BSD
- `-r`: Use extended regular expressions (GNU-specific alias for -E)
- `--in-place`: Long form of -i for in-place editing (GNU only)
- `-z`: Separate input lines by NUL characters instead of newlines
