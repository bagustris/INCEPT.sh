## Detailed Flag Reference (from project flag table)

- `-z`: Filter the archive through gzip for compression/decompression
- `-j`: Filter the archive through bzip2 for compression/decompression
- `-J`: Filter the archive through xz for compression/decompression
- `--exclude`: Exclude files matching the given glob pattern from the archive
- `--strip-components`: Strip N leading path components from file names on extraction
- `-C`: Change to the specified directory before performing operations
- `--xattrs`: Save and restore extended attributes when creating or extracting
