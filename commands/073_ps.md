## Detailed Flag Reference (from project flag table)

- `aux`: Display all processes in BSD-style user-oriented format with terminal and CPU/memory stats
- `-ef`: Display all processes in full POSIX format with UID, PID, PPID, and command line
- `--sort`: Sort output by the specified column (prefix with + for ascending, - for descending)
- `-o`: Specify user-defined output format as a comma-separated list of column keywords
- `--forest`: Display process hierarchy as an ASCII art tree
